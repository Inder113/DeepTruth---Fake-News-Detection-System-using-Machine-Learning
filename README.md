# 📰 DeepTruth - Fake News Detection System

## 📌 Overview

DeepTruth is a machine learning-based system designed to detect fake news using Natural Language Processing (NLP) techniques. The project analyses textual data and classifies news articles as **real** or **fake** using supervised learning models.

This project demonstrates an end-to-end machine learning pipeline including data preprocessing, feature extraction, model training, and evaluation.

---

## 🎯 Objectives

* To build a reliable fake news detection system
* To apply NLP techniques for text processing
* To train and evaluate machine learning models
* To analyse model performance and accuracy

---

## ⚙️ Features

* Data cleaning and preprocessing
* Text vectorization (TF-IDF / CountVectorizer)
* Model training (e.g., Logistic Regression / Naive Bayes / etc.)
* Accuracy and performance evaluation
* Prediction on new/unseen data

---

## 🧠 Methodology

The system follows these steps:

1. **Data Collection**

   * Dataset containing real and fake news articles

2. **Data Preprocessing**

   * Lowercasing
   * Removing stopwords
   * Tokenization

3. **Feature Extraction**

   * Converting text into numerical vectors using NLP techniques

4. **Model Training**

   * Applying machine learning algorithms for classification

5. **Evaluation**

   * Measuring accuracy, precision, recall, and F1-score

---

## 📊 Results

The model was evaluated using standard metrics such as:

* Accuracy
* Confusion Matrix
* Precision & Recall

The system demonstrates effective performance in distinguishing fake news from real news.

## ▶️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/inder13/DeepTruth-Fake-News-Detection.git
```

2. Navigate to the project folder:

```bash
cd DeepTruth-Fake-News-Detection
```

3. Install required libraries:

```bash
pip install -r requirements.txt
```

4. Open Jupyter Notebook:

```bash
jupyter notebook
```

5. Run all cells in:

```
Fake_News_Detector_Final.ipynb
```
